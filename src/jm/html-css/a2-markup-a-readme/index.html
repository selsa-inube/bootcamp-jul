React

React is JavaScript library for building user interfaces.

  • Declarative: react makes it painless to create interactive Ul. Desing simple views for each states in your application, and React will efficiently update and render just the right components when your data changes. Declaratives views make your code more predicable, simpler to understand, and easier to debug.
  • Component-Based: Build encapsulated components that manage their own state, then compose them to make complex Uls. Since component logic is written in JavaScript instead of templates, you can easily pass rich data through your app and keep the state out of the DOM.
  • Learn Once, Write Anywhere: We don't make assumptions about the rest of your technology stack, so you can delevop new feature in react woithout rewriting exiting code. React can also render on the server using Node and power mobile apps using React Native
Learn how to us React in you project

intallation

React has been desingned for gradual adoption from the start, and you can us as litlle or as much React as your need:

  • Use Online Playgrounds to get a taste for React.
  • Add React to a website as a script tag in one minnute.
  • Create New React App if you're looking for a powerful JavaScript toolchain.

you can use React as a script tag from a CDN, or as a React package on npm.

this example will render "Hello Taylor" into a container on the page.


you'll notice that we used an HTML-like sintax; we call its JSX. JSX is not required to use React, but it makes code more readable, and writing it feels like writing HTML. if you're using React as a script tag, read this section on integrating JSX; otherwise, the recommended JavaScript toolchains handle it automatically.
